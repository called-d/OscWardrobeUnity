# OscWardrobe (Unity Editor Extension)

Expression Menu に「アバター変更ボタン」を実装するためのエディタ拡張

メニューを作ると、そのエイリアス定義を `%APPDATA%\jp.0xd.osc-wardrobe.app\defs\` に書き出すので
[本体](https://github.com/called-d/osc-wardrobe) 側から読み取ることができる

## How To Install

### called_D の VPM リポジトリを追加
[![Add To VCC](https://github.com/user-attachments/assets/bfb0d09e-e815-4228-a1af-d23e8ced628b)](vcc://vpm/addRepo?url=https://called-d.github.io/vpm-listing/index.json)

または VCC / ALCOM で以下を追加してください

```
https://called-d.github.io/vpm-listing/index.json
```

### VCC から Osc Wardrobe を追加
アバタープロジェクトに `Osc Wardrobe` パッケージを追加してください
