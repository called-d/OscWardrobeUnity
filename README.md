# OscWardrobe (Unity Editor Extension)

Expression Menu に「アバター変更ボタン」を実装するためのエディタ拡張

メニューを作ると、そのエイリアス定義を `%APPDATA%\jp.0xd.osc-wardrobe.app\defs\` に書き出すので
[本体](https://github.com/called-d/osc-wardrobe) 側から読み取ることができる

## How To Install

### called_D の VPM リポジトリを追加
<a href="vcc://vpm/addRepo?url=https://called-d.github.io/vpm-listing/index.json">

